# Bank Account

This exercise asked us to create a Bank Account class structured in this manner:

| **BankAccount** |
| --------------- |
| - owner : string |
| - balance : int |
| + BankAccount(owner : string) |
| + BankAccount(owner : string, balance : int) |
| + deposit(amount : int) : void |
| + withdraw(amount : int) : void |
| + getOwner() : string |
| + getBalance() : int |
