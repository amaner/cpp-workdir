# Pizza

This project asked us to construct a Pizza class structured in the following manner:

| **Pizza** | 
| --------- |
| - name : string |
| - cost : int |
| - diameter : int |
| - toppings : vector<string> |
| + Pizza(name : string, cost : int, diameter : int) |
| + addTopping(topping : string) : void |
| + getCost() : int |
| + getName() : string |
| + getDiameter() : int |
| + printToppings() : void |
